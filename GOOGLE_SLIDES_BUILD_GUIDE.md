# AthleteIQ Investor Deck — Google Slides Building Guide

## Quick Start

1. Create new blank presentation in Google Slides
2. Set slide size: File → Page setup → Custom (16:9 widescreen)
3. Apply master theme: Blank theme, then customize colors
4. Follow slide-by-slide guide below
5. Copy text exactly as provided

---

## MASTER SLIDE SETUP

### Colors
- **Primary Dark:** #080808 (text, borders)
- **Accent Gold:** #F59E0B (highlights, accents)
- **White/Light:** #FFFFFF, #F9F9F9 (backgrounds)
- **Gray Text:** #333333, #666666 (secondary text)

### Typography
- **Headlines:** Google Font "Barlow Condensed" Bold 900
- **Body Text:** Google Font "Barlow" Regular 400
- **Data/Numbers:** Google Font "Roboto Mono" 700

### Slide Layout
- Padding: 60px from all edges
- Line height: 1.2 for headlines, 1.6 for body
- Background: White or light gradient

## GOOGLE FONTS TO INSTALL

Add these fonts to your presentation before building:
1. Barlow Condensed (700, 800, 900 weights)
2. Barlow (400, 600 weight)
3. Roboto Mono (700 weight)

**How to add:**
1. Click "Theme" in Google Slides menu
2. Click "Customize theme"
3. Click font dropdown
4. Search "Barlow Condensed" → Add
5. Repeat for Barlow and Roboto Mono

## SHARING & PRESENTING

**For remote presentations:**
- Use Google Slides' built-in presenter view
- Test video/audio beforehand
- Have PDF backup on your laptop

**For in-person presentations:**
- Arrive early, test projection
- Use presenter remote to advance slides
- Have printed 1-pager backup

**For email sharing:**
- Use "Share" button in Google Slides
- Set to "Viewer" access (not editable)
- Create PDF export (File → Download → PDF) for archival
