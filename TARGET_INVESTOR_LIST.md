## RED FLAGS TO AVOID

When pitching, be ready for these questions:

1. **"How do you get data at scale?"**
   - Answer: Operations access = data capture. No surveys, no integrations, direct feed.

2. **"What if a bigger player (Nike, Strava) enters?"**
   - Answer: We own the operations layer first. Data advantage comes from operational dominance.

3. **"How do you prevent churn?"**
   - Answer: 1) High switching costs (historical data lock-in), 2) Sticky operations, 3) Value from intelligence layer.

4. **"Why hasn't this existed yet?"**
   - Answer: Data wasn't valuable enough to justify ops complexity. AI/ML now makes data actionable.
